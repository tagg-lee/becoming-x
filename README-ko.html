Becoming X

Quartz 4 기반 디지털 가든

소개

Becoming X는 지식 공유와 지속적인 학습을 위한 개인 디지털 가든입니다. Quartz 4 정적 사이트 생성기로 구축되었으며, Obsidian에서 작성한 콘텐츠를 아름답고 상호 연결된 웹사이트로 발행합니다.

기술 스택

분류기술
프레임워크Quartz 4.5.2
콘텐츠Obsidian (심볼릭 링크된 볼트)
렌더링Preact
스타일링SCSS + LightningCSS
빌드esbuild
호스팅GitHub Pages
도메인becoming.tagglee.me

페이지 레이아웃

콘텐츠 페이지 (개별 노트):

  • 왼쪽 사이드바: 페이지 제목, 검색, 다크 모드, 리더 모드, 탐색기
  • 오른쪽 사이드바: 그래프, 목차, 백링크
  • 본문 상단: 브레드크럼, 글 제목, 메타데이터, 태그

목록 페이지 (태그/폴더):

  • 왼쪽 사이드바: 페이지 제목, 검색, 다크 모드, 탐색기
  • 오른쪽 사이드바: (없음)

주요 기능

  • SPA 내비게이션 - 빠르고 부드러운 페이지 전환
  • 링크 팝오버 - 위키피디아 스타일의 호버 미리보기
  • 전체 텍스트 검색 - FlexSearch 기반
  • 인터랙티브 그래프 - 연결 시각화 (D3 + Pixi.js)
  • 다크/라이트 모드 - 시스템 설정 연동 토글
  • 리더 모드 - 집중 읽기 환경
  • RSS 피드 - 업데이트 구독
  • 사이트맵 - SEO 친화적 구조
  • 수식 렌더링 - KaTeX 지원
  • 구문 강조 - GitHub 테마 코드 블록

배포 워크플로우

sequenceDiagram
    participant Dev as 개발자
    participant Local as 로컬 머신
    participant GH as GitHub
    participant Pages as GitHub Pages

    Dev->>Local: npm run deploy
    Local->>Local: public/에 사이트 빌드
    Local->>GH: gh-pages 브랜치에 푸시
    GH->>Pages: deploy.yml 워크플로우 트리거
    Pages->>Pages: becoming.tagglee.me에 배포
